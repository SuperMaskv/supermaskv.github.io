### MacType

### Listary

### MouseInc

全局鼠标手势，解决手需要频繁在键盘和鼠标间移动的问题，提高操作效率。

```javascript
// 下一个桌面
[ [ "SendKeys", "Ctrl+Win+Right" ] ]
// 上一个桌面
[ [ "SendKeys", "Ctrl+Win+Left" ] ]
```
