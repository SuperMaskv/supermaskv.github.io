                # gerardog/gsudo

                gerardog/gsudoopen in new window 是一个可以在 Windows Powershell 中使用管理员身份运行命令的工具，不需要额外以管理员身份打开一个 Powershell，提高操作效率，使用方式与 *nix 系统中的 sudo 工具类似。
